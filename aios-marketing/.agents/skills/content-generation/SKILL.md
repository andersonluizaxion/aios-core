# Content Generation Skill

Skill compartilhada entre Copywriter e Designer para criacao de conteudo de marketing.

## Quando Usar

- Ao criar copies para anuncios (qualquer plataforma)
- Ao definir specs de criativos por plataforma
- Ao escolher framework de persuasao para uma campanha
- Ao adaptar conteudo para diferentes formatos e canais

## Frameworks de Persuasao

Referencia completa em `references/copy-frameworks.md`.

| Framework | Melhor Para | Estrutura |
|-----------|-------------|-----------|
| AIDA | Awareness, primeiro contato | Attention → Interest → Desire → Action |
| PAS | Dor forte, urgencia | Problem → Agitate → Solve |
| BAB | Transformacao, antes/depois | Before → After → Bridge |
| PASTOR | Vendas longas, webinars | Problem → Amplify → Story → Transformation → Offer → Response |
| FAB | Produtos tecnicos, features | Features → Advantages → Benefits |

| Plataforma | Formato Principal | Headline | Body | Imagem |
|------------|-------------------|----------|------|--------|
| Meta Feed | 1:1 ou 4:5 | 40 chars | 125 chars (primary) | 1080x1080 ou 1080x1350 |
| Meta Stories | 9:16 | overlay | overlay | 1080x1920 |

| YouTube In-Stream | 16:9 | N/A | N/A | video |
| YouTube Thumbnail | 16:9 | overlay | N/A | 1280x720 |

## Principios de Copy para Ads

1. **Hook primeiro** - Capturar atencao nos primeiros 3 segundos (video) ou 5 palavras (texto)
2. **Um beneficio por vez** - Nao tentar comunicar tudo em um unico anuncio
3. **Linguagem do publico** - Usar as palavras que o publico usa, nao jargao tecnico
4. **CTA especifico** - "Compre agora com 20% off" > "Saiba mais"
5. **Prova social** - Numeros, depoimentos, resultados concretos
6. **Escassez real** - Urgencia genuina, nunca fabricada
7. **Mobile-first** - 80%+ do trafego e mobile, copy deve funcionar em tela pequena

## Compliance por Plataforma

### Meta Ads
- Texto em imagem: max 20% da area (recomendacao, nao mais regra dura)
- Proibido: claims de saude sem comprovacao, before/after enganoso
- Proibido: simulacao de interface (fake buttons, fake notifications)
- Categorias especiais: housing, credit, employment, social issues

### Google Ads
- Proibido: superlatives sem comprovacao ("o melhor", "numero 1")
- Proibido: pontuacao excessiva ou CAPS LOCK
- Trademark restrictions aplicam
- Landing page deve corresponder ao anuncio

### YouTube Ads
- Primeiros 5 segundos sao criticos (skip button)
- Legendas obrigatorias (85% assiste sem som)
- Proibido: conteudo chocante ou sensacionalista no hook
